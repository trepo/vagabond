Class: GraphQuery

GraphQuery

A Graph Query.

new GraphQuery(graph)

Create a new GraphQuery. Note that all filters are ANDed together.

Parameters:
Name Type Description
graph Graph

The Graph instance.

Methods

edges()

Execute the query and return all of the edges that match. Note that this function is a ES6 generator.

filter(func)

Pass an optional function to use for filtering. It will be called with function(properties, id, label). The function must return true synchronously for the element to be included.
